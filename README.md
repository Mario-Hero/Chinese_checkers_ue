# Chinese Checkers UE 虚幻跳棋

A Chinese checkers game made with UE 5.1.

基于虚幻引擎 5.1的跳棋游戏。

Implement AI using simple BFS and pruning.

使用简单的广度优先搜索和剪枝实现AI.

Only support local play.

仅支持本地对战。
